# Dubai Property Sales — EDA and Data Pipeline Analysis

An end-to-end ETL pipeline and exploratory data analysis (EDA) project on **81,601 Dubai property sale transactions** from the first half of 2023, built with Python and pandas.

---

## Project Overview

This project transforms a raw Dubai property sales CSV file into a clean, analysis-ready dataset and explores two key questions:

1. **How does property subtype relate to sale amount?**
2. **How does location (area) relate to sale price variation?**

The pipeline covers the full Extract → Transform → Load cycle, followed by descriptive visualisations to uncover market patterns across property types and geographic areas.

---

## Key Findings

- **Flats dominate by volume**, accounting for 67.6% of unique transactions — but they are not the highest-value subtype
- **Commercial, Residential, and Villa** properties show the highest and most varied sale amounts
- Specialised subtypes (Residential Flats, Industrial, General Use) carry the highest median prices — AED 13.5M, 10.1M, and 9.25M respectively
- **Jumeirah Village Circle** (9.5%) and **Business Bay** (7.0%) lead in transaction volume, but premium areas like **Al Ras, Marsa Al Arab, Jumeirah Bay, and Bluewaters** command far higher price-per-sqm
- High-volume areas ≠ high-value areas: Dubai's property market is clearly segmented by location

## ETL Pipeline

### Extract — Data Assessment

Issues identified across 19 findings, categorised by risk:

- **Data types** — `transaction_date` stored as object; `is_free_hold` and `parking` flagged for review
- **Missingness** — 211,902 missing cells (11.8%); `master_project` at 99.9% missing (high risk); moderate missingness in `room`, `parking`, `nearest_metro/mall/landmark`, `project`
- **Uniqueness** — 808 exact duplicate rows; 5,753 duplicate `transaction_number` values (some valid multi-record transactions)
- **Categorical quality** — inconsistent room labels (`"1 B/R"` vs `"1 bedroom"`); parking column contained slot codes rather than yes/no values; whitespace contamination across text fields
- **Outliers** — 6,358 outliers in `amount`; 11,015 in `property_size_sqm`; no negative/impossible values found

### Transform — Cleaning Steps

| Problem | Action |
|---|---|
| `transaction_date` stored as string | Converted to `datetime64[ns]` |
| 808 exact duplicate rows | Removed |
| Whitespace in text columns | Stripped and title-cased |
| Inconsistent `room` labels | Standardised (e.g. `"1 B/R"` → `"1 Bedroom"`) |
| `parking` contained slot codes | Replaced with `"Has Parking"` / `"No Parking"` |
| `master_project` at 99.9% missing | Column dropped |
| Moderate missingness in categorical columns | Filled with `"NA"` to preserve records |
| Missing `property_sub_type` linked to Land records | Filled with `"Land"` |
| `transaction_size_sqm` 96% identical to `property_size_sqm` | Used to fill gaps, then dropped |
| Extreme outliers in `amount` and `property_size_sqm` | Capped at upper fence (not deleted) |
| No derived price metric existed | Created `price_per_sqm = amount / property_size_sqm` |

### Load

The cleaned dataset was saved as `dubai_property_sales_cleaned.csv` with:
- Zero exact duplicates
- Minimal remaining missingness (1 value in `property_size_sqm`)
- All key analytical variables in usable formats

## Exploratory Analysis

### Property Subtype and Sale Amount

Transaction volume and transaction value tell very different stories:

- **Flat** — dominates volume (67.6% of transactions) but sits in the mid-to-lower price range
- **Commercial** — highest median sale amount and widest price spread in the top 10
- **Residential Flat, Industrial, General Use** — highest median prices among all subtypes
- Upper outliers across all subtypes reflect Dubai's active luxury segment

A log-scale boxplot was used to handle the skewed price distribution across subtypes.

### Area and Sale Price Variation

- Transactions are spread across **245 areas**, but the top areas dominate activity
- High-volume areas: Jumeirah Village Circle (9.5%), Business Bay (7.0%), Dubai Marina (4.5%)
- Highest median sale amount areas: Al Hebiah Fourth (AED 8.8M), Wadi Al Safa 3 (AED 8.04M), Jumeirah Islands (AED 7.85M)
- Highest price-per-sqm areas: Al Ras, Marsa Al Arab, Al Dhagaya, Island 2, Jumeirah Bay
- Lowest price-per-sqm areas: Lehbab First, Grayteesah, Nazwah, Um Esalay

Only areas with more than 100 transactions were included in the area-level analysis to ensure statistical reliability.

## Limitations

- `transaction_number` is not a fully reliable primary key — some duplicate numbers represent valid multi-record transactions (e.g. multiple buyers or sellers), not data errors
- Outliers in `amount` and `property_size_sqm` were capped rather than removed, which may slightly compress the upper range of summary statistics
- The dataset covers only January–June 2023 and may not reflect full-year or longer-term market trends
- Bus/subway-style location matching was not possible — area analysis is text-based only, so results depend on consistent area naming
